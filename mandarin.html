Mandarín

Vocabulario y estructuras gramaticales fundamentales.


1. Pronombres personales (我, 你, 他, 她, 我们, 你们, 他们)


2. Pronombres interrogativos (什么, 谁, 哪, 哪儿, 几, 多少, 怎么样)


3. Numerales y contadores (一, 二, 三... 个, 本, 岁)


4. Estructura de oraciones afirmativas (SVO: 我吃饭)


5. Estructura de oraciones negativas (不 + verbo: 我不吃肉)


6. Estructura de oraciones interrogativas con 吗 (你喜欢猫吗？)


7. Estructura de oraciones con 是 (我是学生)


8. Oraciones con 有 para posesión (我有一本书)


9. Oraciones con 在 para ubicación (他在家)


10. Uso de 了 para acciones completadas (我吃饭了)


11. Uso de 很 para adjetivos (她很漂亮)


12. Uso de 太…了 para énfasis (太好了！)


13. Uso de 的 para posesión (我的书)


14. Uso de 在 para acciones en progreso (我在看书)


15. Uso de 想 para expresar deseo (我想喝水)


16. Uso de 喜欢 para expresar gustos (我喜欢音乐)


17. Adverbios de tiempo (今天, 昨天, 明天, 现在)


18. Adverbios de frecuencia (经常, 有时候, 从不)


19. Uso de 多少 y 几 para preguntar cantidad (你有多少个苹果？)


20. Uso de 呢 para preguntas contextuales (你呢？)




1. Expresiones con 要 (querer/necesitar): 我要喝咖啡。


2. Verbos modales como 能, 可以 (poder): 我能说中文。


3. Uso de 正在 para acciones en curso: 我正在吃饭。


4. Comparaciones con 比: 他比我高。


5. Estructura con 因为…所以… (porque… por eso…): 因为下雨了，所以我没去。


6. Expresiones de tiempo (以前，以后，的时候): 吃饭以后，我去工作。


7. Preguntas con 怎么 (cómo): 你怎么去学校？


8. Preguntas con 为什么 (por qué): 你为什么学习汉语？


9. Uso de 过 para experiencias pasadas: 我去过中国。


10. Expresiones con 从…到… (de… a…): 我从早上到晚上都在工作。


11. Uso de 要…了 para indicar inminencia: 火车要开了。


12. Preguntas alternativas con 还是: 你喝茶还是咖啡？


13. Repetición del verbo para suavizar tono (试试，看看): 你试试这个菜。


14. Uso de 把 (inicio de introducción a esta estructura): 把门打开。


15. Intensificadores: 更（más）y 最（el más）: 他更喜欢猫。


16. Uso de 每 para hablar de frecuencia: 我每天六点起床。


17. Adverbios como 已经, 就, 才: 他已经走了。


18. Expresiones de dirección con 往, 到: 往左走，到学校了。


19. Oraciones con resultado (verbo + 到/见/好): 我找到了书。


20. Estructura 一边…一边… para acciones simultáneas: 我一边听音乐一边学习。




1. Estructuras condicionales con 如果…就…: 如果下雨，我们就不去。


2. Uso avanzado de 把: 把衣服洗了。


3. Complementos de resultado y dirección: 说完了，走出去。


4. Uso de 除了…以外，还…: 除了中文以外，我还会说英文。


5. Frases con 虽然…但是…: 虽然很难，但是我喜欢。


6. Uso de 还是…吧 para sugerencias: 我们还是走吧。


7. Preguntas con 多长时间, 多远: 从这里到机场多远？


8. Estructura con 连…都…: 他连汉字都不认识。


9. Verbos dobles (趋向补语): 走进来，跑出去。


10. Uso de 被 para voz pasiva: 我的手机被人拿走了。


11. Partícula 地 para modificar verbos: 他高兴地笑了。


12. Reforzadores: 一点儿, 有点儿, 非常


13. Revisión de cuantificadores y clasificadores: 两条裤子，一杯水


14. Uso de 还是 como aún (todavía): 他还是不来。


15. Verbos seriales: 我去商店买东西。


16. Estructuras con 先…然后…: 我先吃饭，然后去工作。


17. Uso de 越…越…: 天气越来‍越冷。


18. Verbos compuestos comunes: 参加，打算，影响，关心


19. Frases con 一…就… (tan pronto como…): 我一回家就吃饭。


20. Vocabulario temático: salud, viajes, trabajo, familia, clima, escuela.




1. Uso de 即使…也… (aunque… aún así…): 即使下雨，我也要去。


2. Uso de 只要…就… (siempre que… entonces…): 只要你努力，就会成功。


3. Uso de 不仅…而且… (no solo… sino también…): 他不仅会唱歌，而且跳舞跳得很好。


4. Partícula 被 en estructuras más complejas: 他被老师表扬了。


5. Uso de 为了… (con el fin de): 为了健康，我每天跑步。


6. Estructuras de comparación avanzada: A跟B一样 / 没有A那么B


7. Expresiones causales con 由于… (debido a): 由于天气不好，比赛取消了。


8. Uso de 看起来, 听起来, 好像 para suposiciones: 他好像生病了。


9. Preguntas retóricas y exclamaciones: 你说我能不生气吗？


10. Expresiones con 否则 (de lo contrario): 你得早点走，否则要迟到了。


11. Conectores formales: 首先, 其次, 最后


12. Complementos de grado: 跑得很快，写得很好


13. Estructuras con 被, 让, 叫 para voz pasiva o causativa: 我被叫去开会了。


14. Verbos con 着 para estado continuo: 门开着呢。


15. Estructura con 一方面…另一方面…: 一方面我想工作，另一方面我想旅行。


16. Uso de 差不多, 大概, 可能, 应该 para expresar estimación o probabilidad


17. Frases con 既然…就… (ya que… entonces…): 既然你不喜欢，就不要吃了。


18. Palabras como 顺便, 尽管, 竟然, 原来


19. Modificadores avanzados: 那些穿红衣服的人是学生。


20. Estructuras condicionales y finales: 要是你早点来，我们就一起走了。




1. Frases con 无论…都… (no importa… igual…): 无论你怎么说，我都不答应。


2. Uso de 不是…而是… (no es… sino…): 他不是不会，只是不想做。


3. Expresiones con 尽管…还是…: 尽管他很累，还是坚持工作。


4. Estructuras condicionales con 只要/哪怕/即便…也…


5. Estructuras con 一旦…就… (una vez que…): 一旦开始，就不能停止。


6. Frases con 宁可…也不… (preferiría… antes que…): 我宁可晚点，也不想错过。


7. Uso avanzado de 被, 让, 叫, 使


8. Uso de modismos comunes: 马马虎虎，三心二意，对牛弹琴


9. Estilo indirecto: 他说他已经看过那部电影了。


10. Frases con 凡是…都…: 凡是喜欢学习的人，都有进步。


11. Expresiones formales: 例如，此外，总之，然而


12. Uso de 据说, 据…报道, 听说


13. Uso de 各, 每, 所有, 任何 correctamente


14. Frases con 既…又…: 他既聪明又努力。


15. Verbos reduplicados para suavidad o cortesía: 看看，说说，聊聊


16. Uso de 似的, 好像, 仿佛 para comparaciones indirectas


17. Estructuras complejas: 之所以…是因为… / 之类的 / 像…这样/那样的


18. Frases condicionales con “否则”、“要不然”


19. Uso de 成语 y expresiones idiomáticas básicas en contexto


20. Capacidad para escribir composiciones estructuradas con introducción, desarrollo y conclusión.




1. Dominio de todos los conectores discursivos: 尽管…, 不过…, 因此…, 从而…, 以免…, 以致…, 反之…


2. Uso fluido de 成语, refranes, citas clásicas: 塞翁失马，焉知非福


3. Matices en el uso de partículas y modismos regionales


4. Dominio de la voz pasiva y causativa con estructuras poco comunes


5. Uso de estructuras literarias y formales: 倘若…, 若非…, 恐怕…, 未免…


6. Dominio de sinónimos con matices distintos: 看 vs. 观察 vs. 审视


7. Comprensión de ironía, sarcasmo, lenguaje figurado


8. Capacidad para argumentar en ensayos: 提出问题 -> 分析问题 -> 解决问题


9. Redacción en diferentes registros: emails formales, reportes, opiniones


10. Análisis de textos clásicos, poesía, filosofía china


11. Lectura rápida con captación de ideas clave


12. Traducción chino-español y viceversa con precisión


Argumentación oral y debate: 引用数据，列举例子，提出反例


14. Uso activo de frases como: 由此可见，不得不说，归根到底


15. Modulación del tono según contexto: 客气语气，委婉语气，正式语气


16. Manejo de ambigüedades, polisemia y expresiones coloquiales


17. Estructuras causales complejas: 以致于…, 正因为…, 之所以…是由于…


18. Lectura crítica de medios, noticias, artículos de opinión


19. Uso de 语气助词 como 吧, 呢, 呀, 嘛 para matices conversacionales


20. Expresión de emociones complejas y abstractas: 失落，矛盾，感慨，焦虑，充实
