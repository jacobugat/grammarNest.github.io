2. Verbos y conjugaciones

Forma diccionario (辞書形 - jishokei) nivel N4



Forma ない (nai) (Negativa informal)



Forma た (ta) (Pasado informal)



Forma て (te) (Conexión de frases, favores, etc.)



Forma たい (tai) (Deseo)



Forma たことがある (Experiencia pasada)



Forma potencial (られる / える / できる) (Poder hacer algo)



Forma pasiva (れる / られる) (Ser afectado por una acción)



Forma causativa (せる / させる) (Hacer que alguien haga algo)



Forma causativa-pasiva (させられる) (Ser obligado a hacer algo)



Forma volitiva (よう) (Intención, "vamos a...")



Forma condicional (ば / たら / と / なら) (Si... entonces...)



Forma imperativa (なさい / ろ / よ)



Forma prohibitiva (～てはいけない, ～ちゃだめ)



Forma de cortesía (～ください, ～なさい, お～ください)




3. Adjetivos y conjugaciones

Forma adverbial de adjetivos い (～く)



Forma adverbial de adjetivos な (～に)



Comparación (A は B より, A のほうが B より)



Superlativo (いちばん, もっと, ずっと)



Expresión de cambio de estado (～くなる / ～になる)




4. Expresiones y estructuras comunes

～と思います / ～でしょう (Opinión / Probabilidad)



～と言います / ～そうです (Citas directas e indirectas)



～てみる (Intentar hacer algo)



～ておく (Dejar hecho algo de antemano)



～てしまう (Terminar completamente, lamentar algo)



～てある (Estado resultante de una acción)



～ても (Incluso si, aunque)



～ていく / ～てくる (Cambio en el tiempo, acercamiento / alejamiento)



～やすい / ～にくい (Fácil / difícil de hacer algo)



～すぎる (Exceso, demasiado)



～かもしれない (Probabilidad, "quizás")



～そうだ (Parecer que algo va a pasar / rumor)



～はずだ (Debería ser, se supone que)



～でしょう / だろう (Suposición, "probablemente")



～ようだ / ～みたいだ (Parece que, similar a)



～ことにする / ～ことになる (Decidir hacer algo / Se decidió que)



～つもり (Intención de hacer algo)



～場合は (En caso de que)



～たほうがいい / ～ないほうがいい (Es mejor hacer / no hacer algo)



～なければならない / ～なくてはいけない (Obligación, "debo")



～なくてもいい (No es necesario hacer algo)



～てもいいですか？ (Pedir permiso)



～てはいけません (Prohibición, "no se permite")



～てもかまわない (No importa si se hace algo)



～でしょうか？ (Pregunta formal / educada)
